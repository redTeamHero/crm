How to Set Up Facebook Integration

1
Create a Meta Developer App

Go to developers.facebook.com/apps → Create App → Choose "Business" type → Complete setup.

2
Enable Pages API Permissions

In your app dashboard, go to App Review → Permissions → Add: pages_manage_postspages_read_engagementpages_show_list. For testing you can use your own page without review.

Add Credentials to Evolv

Go to Settings → APIs and add your FB_APP_ID and FB_APP_SECRET environment variables. Once added, the Connect button above will activate.

5
Connect Your Page

Click "Connect Facebook Page" above, authorize the app, and select your Business Page. The app will securely store your page access token for publishing.
